# OpenStack Infrastructure Monitoring

## Project Overview

This project presents the design and implementation of a monitoring solution for a private cloud infrastructure based on OpenStack.

The objective is to improve the availability, performance, and reliability of cloud services by collecting, visualizing, and analyzing infrastructure metrics in real time.

This project was completed as part of my Bachelor's degree in Network and Systems Administration.

---

## Objectives

- Monitor OpenStack infrastructure
- Collect system and service metrics
- Visualize real-time performance
- Detect failures quickly
- Improve infrastructure availability
